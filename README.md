# Smart Expense Tracker (Python CLI Project)

## Overview
Smart Expense Tracker is a simple command-line Python application that helps users manage and analyze their daily expenses. It allows you to store expenses, view them, and generate useful insights like monthly spending and category-wise analysis.

---

## Features
- Add new expenses (date, category, amount, description)
- View all saved expenses
- Monthly expense summary
- Category-wise spending analysis
- Pie chart visualization of expenses
- Data stored in JSON format (persistent storage)

---

##  Technologies Used
- Python
- JSON (for data storage)
- Matplotlib (for visualization)
